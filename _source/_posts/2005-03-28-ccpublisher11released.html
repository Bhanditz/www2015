After many, many delays, I'm proud to announce the release of ccPublisher 1.1. This is primarily a bug fix release that corrects the most common problems we've had reported. This release also accounts for changes in the upload requirements at the Internet Archive.

You can find packages for Windows and Mac OS X here. Go on, give it a try. And if you have suggestions or feedback, email me at nathan@creativecommons.org or file a bug.
