I'm super proud to announce that Google Code's Project Hosting now supports, in complimentary addition to software licenses for code, content licensing. Two options available by default are a CC Attribution and CC Attribution-ShareAlike 3.0 license. We've been working on this together for a year or so behind the scenes, so its fabulous to see this come out now. Also, this is especially cool coming right after the release of the brilliant Radiohead "House of Cards" video on Google Code which features Creative Commons licensed animation data available now through Google Code Project Hosting.

And, just to be clear, Google Code Project Hosting is supporting Creative Commons licensing for non-source code assets. This functionality compliments the still supported same software licenses. If one wants to override the default options for setting the content license, this may manually changed by the admins of a project.

DeWitt Clinton, who led this effort at Google said more at the Google Code blog:

While this may seem like a minor change, it reflects the continued evolution of our mission to support best practices in open source software development. As the open web increasingly relies on protocols and formats that reach beyond source code, we encourage authors to apply an explicit copyright license to the data, documentation, and related media that complements their work.

Its nice to see that Google continues to pioneer by supporting the open web. We at CC are hopeful that more and more companies will adopt Creative Commons license support to empower the free and open web that is well beyond just source code. We have free tools which make it easy to to add CC license support, lots of Case Studies to help you decide why to add CC licensing, information documents to explain how you might add CC licensing support and people on the ground to help you through any stage of the process and help promote your project's addition of CC licensing through our various communication channels like this blog :)

A big cheer to our friends at Google who led this effort. Please go host all or part of your project on Google Code Project Hosting, test the feature out and let all know how it works.
