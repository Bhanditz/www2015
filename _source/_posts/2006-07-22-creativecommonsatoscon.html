I'm heading to Portland, Oregon for O'Reilly Media's Open Source Convention (OSCON) tomorrow. I'll be presenting an updated and expanded version of my PyCon talk on Building Extensible Desktop Applications with Zope 3. The more I work with it, the more I realize that a better title would be Loosely Coupled, Component Based Development with Zope 3. So if you're attending OSCON and curious about writing applications as a set of much smaller pieces using Python, stop by. Slides, example code and additional resources are available in the wiki on the talk page. See you there!
